## 📌 معرفی

**Time-Pass-Calculator** برنامه‌ای است که به شما می‌گوید از یک **تاریخ و ساعت خاص در گذشته** چقدر زمان گذشته است، یا تا یک **تاریخ و ساعت خاص در آینده** چقدر زمان مانده است.

این برنامه برای موارد زیر بسیار مفید است:

| کاربرد | مثال |
|--------|------|
| 🎂 **محاسبه سن** | "چند سال، ماه و روز از تولدم می‌گذرد؟" |
| 💍 **سالگرد ازدواج** | "چند سال از زندگی مشترکمان می‌گذرد؟" |
| 💼 **سابقه کاری** | "چند سال در این شرکت کار کرده‌ام؟" |
| 🎓 **فارغ‌التحصیلی** | "چند سال از دانش آموختگی من می‌گذرد؟" |
| ✈️ **سفر بعدی** | "چند روز تا سفر بعدی مانده است؟" |
| 📅 **تولد بعدی** | "چند روز تا تولد بعدی من مانده؟" |

## ✨ ویژگی‌ها

| شماره | ویژگی | توضیح |
|-------|-------|-------|
| 1️⃣ | **انتخاب تاریخ** | انتخاب روز، ماه و سال با DateTimePicker |
| 2️⃣ | **انتخاب ساعت** | انتخاب ساعت و دقیقه دقیق |
| 3️⃣ | **پشتیبانی از گذشته** | اگر تاریخ در گذشته باشد → "چقدر گذشته است؟" |
| 4️⃣ | **پشتیبانی از آینده** | اگر تاریخ در آینده باشد → "چقدر مانده است؟" |
| 5️⃣ | **آپدیت لحظه‌ای** | هر ثانیه نتایج به‌روز می‌شوند |
| 6️⃣ | **نمایش کامل** | نمایش روز، ساعت، دقیقه و ثانیه به صورت جداگانه |
| 7️⃣ | **محاسبه سریع** | با یک کلیک، تایمر شروع می‌شود |

## 🚀 نحوه استفاده

### مرحله 1: اجرای برنامه
فایل `TimePassCalculator.exe` را اجرا کنید.

### مرحله 2: انتخاب زمان
- **تاریخ:** از تقویم بازشو روز، ماه و سال را انتخاب کنید
- **ساعت:** ساعت و دقیقه مورد نظر را تنظیم کنید

### مرحله 3: شروع محاسبه
دکمه **"شروع محاسبه"** را کلیک کنید.

### مرحله 4: مشاهده نتیجه
- اگر تاریخ انتخاب شده **در گذشته است** → "X روز و X ساعت گذشته است"
- اگر تاریخ انتخاب شده **در آینده است** → "X روز و X ساعت مانده است"
